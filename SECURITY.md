# Security Policy

## Supported Versions
Only the latest published version is supported with security fixes.

## Reporting a Vulnerability
Please do not open public issues for security reports.

Use one of these private channels:
- GitHub Security Advisory in this repository
- Direct email to project maintainers

Include:
- affected version
- reproduction steps
- impact summary
- suggested fix (if available)

We will acknowledge reports promptly and coordinate disclosure after a fix is available.
